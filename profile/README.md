# Meetless

**Version control for product change.** Watch for scope drift, govern it with Decision Diffs, and propagate the approved truth to every linked artifact.

Meetless is a change-governance layer for product delivery. When scope changes happen in Slack threads, Jira comments, or PRD edits, Meetless turns them into auditable Decision Diffs with required approvals, impact checklists, and automatic propagation, so engineering and QA stop shipping against stale assumptions.
